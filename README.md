## Install

```
yarn global add so-cli
```

## Usage

```
Usage: so <command> <project-name>

Options:
  -v, --version  output the version number
  -h, --help     output usage information

Commands:
  init           create a new project
  list           view template list
  help [cmd]     display help for [cmd]
```
