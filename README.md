# so-cli

根据模版快速生成项目

## Install

```
npm install -g so-cli
```

## Usage

```
Usage: so <command> [project-name]

Options:
  -v, --version  output the version number
  -h, --help     output usage information

Commands:
  init           create a new project
  list           view template list
  help [cmd]     display help for [cmd]
```
